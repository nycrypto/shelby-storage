# Naming conventions

Use unique names to avoid `already exists` errors.

Recommended pattern:

files/<project>/<type>/YYYYMMDD-HHMMSS-<nanosec>-<rand10>.<ext>

Examples:
- files/storage/screenshots/20260310-120005-283713119-acde1234ab.png
- files/storage/text/20260310-120005-283713119-acde1234ab.txt
